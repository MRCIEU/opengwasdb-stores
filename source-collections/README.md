# Summary-Statistics Source Collections

Source Collections describe homogeneous upstream summary-statistics inventories. Each collection has one Source Format and one Source Reader Capability, plus provider, provenance, locations, access constraints, and validation state.

A Source Collection may be a format-homogeneous subset of a provider rather than the whole provider.

When practical, a Source Collection should include a Source Inventory listing discovered upstream Analyses and their locations, checksums, sizes, and basic readiness metadata. Frequently changing sources may keep dated inventory snapshots.
